Question-1: What are the different ways to select an element in the DOM?

There are many ways to select an element in DOM like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector()etc.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML ==> Returns HTML code inside an element (Including Tags).
innerText ==> Returns/sets the visible text only
textContent ==> Returns/sets all text inside an element (including hidden text)

Question-3: What is event delegation in the DOM ?

Instead of attaching event listeners to multiple child elements, you attach a single event listener to their parent, and use event bubbling to handle events for all children.

Question-4: What is event bubbling in the DOM ?

When an event happens on an element, it first runs on that element, then propagates (bubbles) up to its parent elements, all the way to the document.

Question-5: How do you create, add, and remove elements using JavaScript ?

There are some way to do these like to creating element createElement(), to add element appendChild()/add('property'), to remove element remove('property') etc.
